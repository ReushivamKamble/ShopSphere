# ShopSphere – Full Stack MERN E-Commerce Application

ShopSphere is a full-stack MERN e-commerce web application built with modern web technologies and responsive UI design. The platform provides secure authentication, product management, cart functionality, and complete order workflow.

---

## Features

- JWT-based User Authentication & Protected Routes
- Admin Dashboard with Product CRUD Operations
- Product Catalog with Category Filtering
- Shopping Cart & Quantity Management
- Order Placement & Order History
- Responsive UI for Mobile & Desktop
- Product Image Upload Integration
- RESTful API-based Backend Architecture

---

## Installation

## Clone Repository

```bash
git clone https://github.com/ReushivamKamble/ShopSphere.git
```

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
